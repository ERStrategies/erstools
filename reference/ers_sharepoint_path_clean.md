# Clean beginning path and %20s in sharepoint file names for ers_read_sharepoint and ers_write_sharpoint

Clean beginning path and %20s in sharepoint file names for
ers_read_sharepoint and ers_write_sharpoint

## Usage

``` r
ers_sharepoint_path_clean(full_path)
```

## Arguments

- full_path:

  REQUIRED: The path to the file. This can be pasted directly from
  SharePoint:
  https://app.tettra.co/teams/ersknowledge/pages/copying-a-folder-path-from-sharepoint-to-use-in-r

## Value

Your cleaned sharepoint path and the sharepoint drive name
